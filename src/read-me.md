# Craigslist Playwright Java Test Automation Framework

## Overview
This project is an end-to-end UI test automation framework built using:

- Java
- Playwright
- JUnit 5
- Page Object Model (POM)
- Allure Reporting

It tests core functionality of Craigslist housing search including:
- Sorting (price, newest, upcoming)
- Search functionality
- Price validation

---

## How to Run Tests

### Run all tests
```bash
mvn clean test
